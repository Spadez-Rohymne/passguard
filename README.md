-----PassGuard-----
A serverless password strength checker and random password generator, hosted on AWSS3.
Live Demo: https://youneverknowbuck.s3.ap-south-1.amazonaws.com/index.html ---Click here to visit PassGuard


-----What it does-----
-Checks password strength in real time
-Generates secure random passwords
-100% client side — no data is stored or transmitted


-----Tech Used-----
-HTML, CSS, JavaScript
-AWS S3 (static website hosting)
-----What I learned------
-How to host a static website on AWS S3
-How to configure S3 bucket policies
-How serverless architecture works
